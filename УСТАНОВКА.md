# Инструкция по установке и запуску локально

## Вариант 1: Установка Node.js (рекомендуется для разработки)

### Шаг 1: Установите Node.js

1. Перейдите на официальный сайт: https://nodejs.org/
2. Скачайте LTS версию (рекомендуется 20.x или выше)
3. Запустите установщик и следуйте инструкциям
4. При установке убедитесь, что отмечена опция "Add to PATH"

### Шаг 2: Проверьте установку

Откройте новую командную строку (PowerShell или CMD) и выполните:

```bash
node --version
npm --version
```

### Шаг 3: Установите зависимости и запустите проект

```bash
# Перейдите в директорию проекта
cd "C:\Users\HOME\OneDrive\Desktop\портфолио"

# Установите зависимости
npm install

# Запустите dev сервер
npm run dev
```

### Шаг 4: Откройте в браузере

После запуска команды `npm run dev` откройте браузер и перейдите по адресу:

**http://localhost:3000**

## Вариант 2: Использование Docker (если установлен)

Если у вас установлен Docker Desktop:

```bash
# Перейдите в директорию проекта
cd "C:\Users\HOME\OneDrive\Desktop\портфолио"

# Запустите через Docker Compose
docker-compose up -d

# Или соберите и запустите вручную
docker build -t portfolio .
docker run -p 3000:3000 portfolio
```

## Быстрые команды после установки Node.js

```bash
# Установка зависимостей
npm install

# Запуск dev сервера (с hot-reload)
npm run dev

# Сборка для продакшена
npm run build

# Запуск продакшен версии
npm start

# Проверка кода линтером
npm run lint
```

## Решение проблем

### Проблема: npm не найден после установки Node.js

**Решение:** Перезапустите терминал/командную строку после установки Node.js.

### Проблема: Ошибки при установке зависимостей

**Решение:** Убедитесь, что у вас установлена последняя версия Node.js и npm:

```bash
npm install -g npm@latest
```

### Проблема: Порт 3000 уже занят

**Решение:** Измените порт в команде запуска:

```bash
# Windows PowerShell
$env:PORT=3001; npm run dev

# Или создайте файл .env.local с содержимым:
PORT=3001
```
