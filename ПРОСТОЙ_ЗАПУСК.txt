📌 ИНСТРУКЦИЯ:
──────────────

1. Дважды кликните на файл: БЫСТРЫЙ_ЗАПУСК.bat

2. Если Node.js не установлен:
   → Скрипт откроет сайт nodejs.org
   → Скачайте и установите LTS версию
   → Перезапустите БЫСТРЫЙ_ЗАПУСК.bat

3. Если всё установлено:
   → Скрипт установит зависимости (первый раз)
   → Запустит сервер
   → Откроет браузер автоматически

4. Откройте: http://localhost:3000

───────────────────────────────────────────────

🛠️ АЛЬТЕРНАТИВНЫЕ ФАЙЛЫ:
─────────────────────────

• БЫСТРЫЙ_ЗАПУСК.bat  ← САМЫЙ ПРОСТОЙ (рекомендуется)
• start.bat           ← Минимальная версия
• запустить.bat       ← Полная версия с проверками

───────────────────────────────────────────────

❌ ЕСЛИ НЕ РАБОТАЕТ:
────────────────────

1. Проверьте, установлен ли Node.js:
   Откройте командную строку и введите: node --version
   
2. Если не установлен:
   → https://nodejs.org/
   → Скачайте LTS версию
   → Установите
   
3. Перезапустите скрипт

───────────────────────────────────────────────

✅ ГОТОВО! Кликните БЫСТРЫЙ_ЗАПУСК.bat
